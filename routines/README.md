# Routines

A personal repository for organizing and tracking daily, weekly, and monthly routines.

## Structure

```
routines/
├── morning/          # Morning routine checklists and guides
├── evening/          # Evening/wind-down routine checklists
├── weekly/           # Weekly recurring tasks and reviews
├── monthly/          # Monthly goals, reviews, and resets
├── templates/        # Reusable routine templates
└── logs/             # Completed routine logs and reflections
```

## How to Use

1. **Pick a routine** from the relevant folder (morning, evening, weekly, monthly)
2. **Follow the checklist** - each routine has a step-by-step guide
3. **Log your progress** - copy a template from `templates/` into `logs/` and fill it in
4. **Review and adjust** - use the weekly and monthly reviews to refine what works

## Quick Start

- Start with the [Morning Routine](morning/morning-routine.md)
- End your day with the [Evening Routine](evening/evening-routine.md)
- Every Sunday, do your [Weekly Review](weekly/weekly-review.md)
- First of the month, do your [Monthly Review](monthly/monthly-review.md)
