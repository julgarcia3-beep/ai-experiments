# Morning Routine

A structured morning to set the tone for a productive day.

## Checklist

- [ ] Wake up at target time
- [ ] Hydrate - drink a full glass of water
- [ ] Make bed
- [ ] Personal hygiene (shower, brush teeth, skincare)
- [ ] Get dressed
- [ ] Breakfast
- [ ] Review today's calendar and priorities
- [ ] Set top 3 goals for the day
- [ ] Start first focused work block

## Notes

- Aim to complete within 60-90 minutes of waking
- Avoid checking phone/social media before completing the first 3 items
- Adjust items to fit your lifestyle - this is a starting point
